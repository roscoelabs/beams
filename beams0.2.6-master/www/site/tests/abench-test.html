Apache Bench Test Page

This is a test page designed for use with Apache Bench. At your Beams server's command line, you can direct one or more tests to this page's url to gauge the server's performance.

Full documentation regarding the features and usage of Apache Bench is available from the Apache Software Foundation here.

If you just want to run a simple test to get started, here's one the Roscoe Labs team recommends:

ab -n 1000 -c 200 https://yoursite.com/tests/abench-test.html/

This test runs 1,000 requests to your server, with a maximum of 200 running concurrently.
